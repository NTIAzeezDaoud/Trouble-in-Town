Trouble In Town

The Mafia/Werewolf game that can be played in a local network

Download here: Download

To play, one player must open the program and press on host. Then that same players opens the program again and writes a name and the address and port written on the server window. Other players must do the same to join the game. One all players are ready to play press on the Start button and then you are good to go. Be sure to minimise the Server window or else you will see some secret information about the game and the other players.

Be sure to submit any bugs to the GitHub repo!
